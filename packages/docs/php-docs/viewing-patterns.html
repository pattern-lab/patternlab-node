--- title: Viewing Patterns tags: - docs --- Pattern Lab utilizes PHP's [built-in web server](http://php.net/manual/en/features.commandline.webserver.php) to let you browse your generated patterns. To start the server do the following: 1. In a terminal window navigate to the root of your project 2. Type `php core/console --server` Your local Pattern Lab install should now be available for browsing at [http://localhost:8080](http://localhost:8080).
