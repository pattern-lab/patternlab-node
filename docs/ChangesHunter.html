checkBuildState(pattern, patternlab)

Checks the build state of a pattern by comparing the modification date of the rendered output file with the Pattern.lastModified. If the pattern was modified after the last time it has been rendered, it is flagged for rebuilding via CompileState.NEEDS_REBUILD.
