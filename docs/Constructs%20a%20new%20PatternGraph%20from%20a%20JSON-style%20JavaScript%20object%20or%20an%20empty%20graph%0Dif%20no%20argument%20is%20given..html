Constructs a new PatternGraph from a JSON-style JavaScript object or an empty graph if no argument is given.(graph, timestamp, version) → {Object}

new Constructs a new PatternGraph from a JSON-style JavaScript object or an empty graph if no argument is given.(graph, timestamp, version) → {Object}

Wrapper around a graph library to build a dependency graph of patterns. Each node in the graph will maintain a CompileState. This allows finding all changed patterns and their transitive dependencies. Internally the graph maintains a PatternRegistry to allow fast lookups of the patterns.
Parameters:
Name Type Description
graph Graph The graphlib graph object
timestamp int The unix timestamp
version int The graph version.
